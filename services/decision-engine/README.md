# Decision Engine AI Service

AI-powered decision support service that compares alternatives, evaluates trade-offs, and provides recommendations.

## Features

- Alternative evaluation
- Trade-off analysis
- Decision recommendations
- Side-by-side comparison
- Criteria-based scoring
- Context-aware decision making

## API Endpoints

- `POST /evaluate` - Evaluate alternatives and get recommendation
- `POST /compare` - Compare alternatives side-by-side
- `GET /health` - Health check

## Environment Variables

- `DATABASE_URL` - PostgreSQL connection string
- `REDIS_URL` - Redis connection string
- `OPENAI_API_KEY` - OpenAI API key for AI analysis

## Example Use Cases

- Monolith vs Microservices decision
- SQL vs NoSQL selection
- Cloud provider selection
- Technology stack decisions
- Architecture pattern selection
